5. Logout Current Session
    Request
        Method : GET
        Header : Authorization : <token>
        URL : {{url}}/user/logout
        Parameters : none
    Response
        {
            "status": 200,
            "message": "Success",
            "response": []
        }


6. Logout From All Devices
    Request
        Method : GET
        Header : Authorization : <token>
        URL : {{url}}/user/logoutall
        Parameters : none
    Response
        {
            "status": 201,
            "message": "Accepted",
            "response": {
                            "status": 200,
                            "message": "Success",
                            "response": []
                        }
        }
