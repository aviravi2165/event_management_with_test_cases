4. Join Event
    Request
        Method : POST
        Header : Authorization : <token>
        URL : {{url}}/event/join/{{event_id}}
        Parameters : event_id
    Response
       {
            "status": 200,
            "message": "Success",
            "response": "Event Joined Successfully"
        }


5. Leave Event
    Request
        Method : POST
        Header : Authorization : <token>
        URL : {{url}}/event/leave/{{event_id}}
        Parameters : event_id
    Response
       {
            "status": 200,
            "message": "Success",
            "response": "Event left successfully"
        }
